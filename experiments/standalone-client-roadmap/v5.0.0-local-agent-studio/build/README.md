# Application Icon Provenance

## English

`icon.png` was generated specifically for Local Agent Studio on 2026-07-02
with OpenAI image generation. The prompt requested an original abstract local
coding-agent mark and explicitly excluded text, watermarks, third-party logos,
and lookalikes of OpenAI, Codex, ChatGPT, GitHub, Microsoft, and Apple marks.

The source PNG is the cross-platform master. Electron Builder converts it into
the platform-specific Windows, macOS, and Linux icon formats during packaging.

## Tiếng Việt

`icon.png` được tạo riêng cho Local Agent Studio ngày 2026-07-02 bằng công
cụ tạo ảnh OpenAI. Prompt yêu cầu biểu tượng coding agent local nguyên bản và
loại trừ chữ, watermark, logo bên thứ ba hoặc hình tương tự thương hiệu OpenAI,
Codex, ChatGPT, GitHub, Microsoft và Apple.

PNG nguồn là master cross-platform. Electron Builder tự chuyển file này sang
định dạng icon Windows, macOS và Linux trong quá trình đóng gói.
